# Grupo NBG Import - Vite + React

## Cómo usar

1. Descomprime el proyecto.
2. Abre una terminal dentro de la carpeta.
3. Ejecuta:

```bash
npm install
npm run dev
```

## Para subir a Hostinger

### Opción recomendada: hosting estático

1. Ejecuta:

```bash
npm install
npm run build
```

2. Se generará la carpeta `dist`.
3. Sube **todo el contenido de `dist`** a `public_html` en Hostinger.

### Si Hostinger te pide una estructura válida

Este proyecto ya incluye:
- `package.json`
- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `vite.config.js`

Es decir, ya es una estructura válida de Vite + React.
